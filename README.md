### 🌳 프로젝트 구조
```
src/
 ├── assets/                   # 이미지/아이콘
 ├── components/               # 공통 UI 컴포넌트
 ├── pages/                    # 각 페이지 화면
 │     ├── EducationDetails/   # 교육 상세 (메뉴얼/퀴즈)
 │     ├── EducationManagement/  # 교육 생성
 │     ├── Schedule/           # 스케줄
 │     ├── HomePageEmployee/
 │     ├── HomePageOwner/
 │     └── ...
 ├── router/                   # 라우터 설정
 ├── styles/                   # 전역 스타일
 ├── utils/                    # 포맷·헬퍼 함수
 └── App.tsx

```
